# Standards
## Description
This is a collection of scripts that can be used to reproduce the results of the article **On the Limited Generalizability of Specialized Band Importance Functions for Predicting Mandarin Speech Intelligibility**

## Installation
You need a running MATLAB installation, Matlab Audio toolbox and the ITA-toolbox, which can be found under: https://www.ita-toolbox.org/

## How to use
run the scripts named "worksheet_X" from within a MATLAB console
